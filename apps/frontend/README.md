# GitSyntropy Frontend

## Commands

- `npm install`
- `npm run dev`
- `npm run build`

## Environment variables

- `PUBLIC_API_BASE` (default `http://localhost:8000/api/v1`)
- `PUBLIC_WS_BASE` (default `ws://localhost:8000`)
- `PUBLIC_AUTH_REQUIRED` (default `false`; set `true` to re-enable protected routes)
- `PUBLIC_AUTH_BYPASS_USER_ID` (default `user_local` when auth is bypassed)
